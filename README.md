# Detecci-n-de-Spam-en-SMS-con-Naive-Bayes-y-NLP-Bag-of-Words-y-N-gramas-
Proyecto de clasificación de spam en mensajes SMS utilizando técnicas clásicas de NLP (Bag-of-Words, n-gramas, eliminación de stopwords) y un clasificador Multinomial Naive Bayes. Evaluación mediante matriz de confusión y curva Precision-Recall.
